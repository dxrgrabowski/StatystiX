# StatystiX

How to run:

1 Method QC IDE
-Open QT Creator
-Open cloned folder as a project
-Run CMakeLists.txt

2 Method Cmake
-Build as a cmake project

You may run into a problem using a custom font, if the program is built using the first method add AveriaSerifLibre fonts to Windows/fonts dir
